# Kush Gulati - Data Scientist & ML Engineer Portfolio

Welcome to the source code for my professional portfolio website! This repository contains the code for a modern, responsive, and interactive single-page application built to showcase my data science skills, projects, and professional experience.

## 🚀 Features

* **Modern Light Theme:** A clean, professional aesthetic utilizing a carefully selected color palette (indigo/slate accents on white/light-gray backgrounds).
* **Interactive SPA Feel:** Custom JavaScript handles tab navigation (Home, Skills, Internships, Projects, Certifications, Contact, Education) without full page reloads.
* **Responsive Design:** Fully fluid CSS variables, CSS grid/flexbox layouts, and media queries ensure the site looks beautiful on desktops, tablets, and mobile devices.
* **Component-Based UI:** Modular HTML architecture utilizing reusable card components, custom "glassmorphic" hover glow effects, and pill-shaped tags.
* **Infographic Data Rendering:** Extensive use of progress bars, timeline components, and icon grids to visually break down technical competencies and soft skills.
* **Verified Credentials:** Direct external integrations to Coursera and Google Drive for immediate certificate authentication.
* **Contact Integration:** A fully styled form utilizing Formspree for direct email contact from the frontend.

## 🛠️ Technology Stack

This portfolio is built using core web technologies to ensure maximum performance and zero dependency overhead:

* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Custom variables (CSS properties) for robust theming, Flexbox/Grid for layout, and modern CSS animations (`@keyframes` and transitions).
* **Vanilla JavaScript:** DOM manipulation, interactive tab routing, and responsive mobile menu toggling.
* **Formspree:** Backend email API integration for the contact form.

## 💻 Running Locally

To view the portfolio on your local machine, you simply need a local web server to prevent browser CORS issues with local assets.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kushgulati033/portfolio.git
   cd portfolio
   ```

2. **Start a local server:**
   If you have Python installed, you can easily spin up a server:
   ```bash
   python -m http.server 8000
   ```
   *Alternatively, you can use the Live Server extension in VS Code.*

3. **View the site:**
   Open your browser and navigate to `http://localhost:8000`.
